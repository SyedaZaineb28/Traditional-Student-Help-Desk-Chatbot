# Traditional-Student-Help-Desk-Chatbot
A simple rule-based chatbot developed in Python using Google Colab. The chatbot demonstrates traditional chatbot development using keyword matching and regular expressions without using large language models.
# Azure AI Sentiment Chatbot

## Project Overview

This project demonstrates the integration of a traditional Python chatbot with Microsoft Azure AI Language Service.

The chatbot accepts user input, performs sentiment analysis using Azure AI, and displays confidence scores for positive, neutral, and negative sentiment.

## Features

- Traditional rule-based chatbot
- Azure AI Language Service integration
- Sentiment analysis
- Confidence score display
- Help menu
- Empty input validation
- Exit command
- Exception handling

## Technologies Used

- Python
- Google Colab
- Microsoft Azure AI Language Service
- Azure Text Analytics SDK
